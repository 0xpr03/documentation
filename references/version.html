# Version

# Get version of meilisearch

GET
/version

Get version of MeiliSearch.

# Headers

Header Value
X-Meili-API-Key $API_KEY
Accept-encoding gzip, deflate

# Example

curl \
  --location \
  --request GET 'http://localhost:8080/version' \
  --header "X-Meili-API-Key: $API_KEY"
