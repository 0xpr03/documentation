# Stop words

# Get the list of stop-words

GET
/indexes/:index/stop-words

Get the list stop-words.

# Path Variables

Variable Description
index The index UID

# Example

curl \
  --request GET 'http://localhost:8080/indexes/12345678/stop-words'

# Response: 200 Ok

List of all the stop-words in the index.

["of","the","to"]

# Add stop-words

PATCH
/indexes/:index/stop-words

Add stop-words to the list.

# Path Variables

Variable Description
index The index UID

# Body

An array of strings containing the stop-words.

# Example

curl \
  --request PATCH 'http://localhost:8080/indexes/12345678/stop-words' \
  --data '["the", "of", "to"]'

# Response: 202 Accepted

{
  "updateId": 1,
}

This update id allows you to track the current action.

# Delete stop-words

DELETE
/indexes/:index/stop-words

Delete a list of stop-words from the list.

# Path Variables

Variable Description
index The index UID

# Body

An array of strings containing the stop-words to delete.

# Example

curl \
  --request POST 'http://localhost:8080/indexes/12345678/stop-words' \
  --data '["the"]'

# Response: 202 Accepted

{
  "updateId": 1,
}

This update id allows you to track the current action.
