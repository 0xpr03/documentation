# Indexes

# List all indexes

GET
/indexes

List all indexes.

# Headers

Header Value
X-Meili-API-Key $API_KEY
Accept-encoding gzip, deflate

# Example

curl \
  --location \
  --request GET 'http://localhost:8080/indexes' \
  --header "X-Meili-API-Key: $API_KEY"

# Response: 200 Ok

[
  {
    "name": "Movie",
    "uid": "12345678",
    "createdAt": "2019-11-20T09:40:33.711324Z",
    "updatedAt": "2019-11-20T10:16:42.761858Z"
  }
]

# Create an index

POST
/indexes

Create an index.

The schema definition is optionally send through the body. If no schema has been defined when the first document is sent it will be infered based on that document.

A randomly generated UID will be returned. It's associated to the new index. This UID will be essential to make all request over the created index.

# Headers

Header Value
X-Meili-API-Key $API_KEY
Content-Type application/json
Accept-encoding gzip, deflate

# Body

The body take at least an index name.

The body accepts an optional schema definition of your documents.

{
  "name": "Movies",
  "schema": {
    "id": ["identifier", "indexed", "displayed"],
    "title": ["indexed", "displayed"],
    "description": ["indexed", "displayed"],
    "release_date": ["displayed"],
    "cover": ["displayed"]
  }
}

The index can be created without any schema.

If no schema has been defined when the first document is added, the schema will be infered based upon the first document.

# Example

curl \
  --location \
  --request POST 'http://localhost:8080/indexes' \
  --header 'Content-Type: application/json' \
  --header "X-Meili-API-Key: $API_KEY" \
  --data '{
  "name": "Movies",
  "schema": {
    "id": ["identifier", "indexed", "displayed"],
    "title": ["displayed", "indexed"],
    "overview": ["displayed", "indexed"],
    "release_date": ["displayed"],
    "poster": ["displayed"]
  }
}'

If no schema has been defined :

# Response: 201 created

{
  "name": "Movie",
  "uid": "12345678",
  "schema": null,
  "createdAt": "2019-11-20T09:40:33.711476Z",
  "updatedAt": "2019-11-20T09:40:33.711476Z"
}

If schema has been given :

# Response: 201 created

{
  "updateId": 1,
  "name": "Movie",
  "uid": "12345678",
  "schema": {
    "id": ["identifier", "indexed", "displayed"],
    "title": ["displayed", "indexed"],
    "overview": ["displayed", "indexed"],
    "release_date": ["displayed"],
    "poster": ["displayed"]
  },
  "createdAt": "2019-11-20T09:40:33.711476Z",
  "updatedAt": "2019-11-20T09:40:33.711476Z"
}

This update id allows you to track the current action.

# Update an index

PUT
/indexes/:index

Update an index name.

# Headers

Header Value
X-Meili-API-Key $API_KEY
Content-Type application/json
Accept-encoding gzip, deflate

# Path Variables

Variable Description
index The index UID

# Body

The body accepts a new name for the given index.

{
  "name": "Movies"
}

# Example

curl \
  --location \
  --request PUT 'http://localhost:8080/indexes/12345678' \
  --header 'Content-Type: application/json' \
  --header "X-Meili-API-Key: $API_KEY" \
  --data '{
  "name": "Movies"
}'

# Response: 202 Accepted

{
  "name": "Movie",
  "uid": "12345678",
  "createdAt": "2019-11-20T09:40:33.711324Z",
  "updatedAt": "2019-11-20T10:16:42.761858Z"
}

# Delete an index

DELETE
/indexes/:index

Delete an index.

# Headers

Header Value
X-Meili-API-Key $API_KEY
Content-Type application/json
Accept-encoding gzip, deflate

# Path Variables

Variable Description
index The index UID

# Body

no body

# Example

curl \
  --location \
  --request DELETE 'http://localhost:8080/indexes/12345678' \
  --header 'Content-Type: application/json' \
  --header "X-Meili-API-Key: $API_KEY" \

# Response: 200 OK

{
  "updateId": 1,
}

This update id allows you to track the current action.

# Get one index schema

GET
/indexes/:index/schema

Get the schema of one index.

# Headers

Header Value
X-Meili-API-Key $API_KEY
Accept-encoding gzip, deflate

# Path Variables

Variable Description
index The index UID

# Query Parameters

Query Parameter Description Default Value
raw Change the format of response false

# Example

curl \
  --location \
  --request GET 'http://localhost:8080/indexes/12345678/schema' \
  --header "X-Meili-API-Key: $API_KEY"

# Response: 200 Ok

{
  "id": ["identifier", "indexed", "displayed"],
  "title": ["displayed", "indexed"],
  "overview": ["displayed", "indexed"],
  "release_date": ["displayed"],
  "poster": ["displayed"]
}

If raw query parameter has been set to true :

{
  "identifier": "id",
  "attributes": {
    "id": {
      "displayed": true,
      "indexed": true,
      "ranked": false
    },
    "title": {
      "displayed": true,
      "indexed": true,
      "ranked": false
    },
    "overview": {
      "displayed": true,
      "indexed": true,
      "ranked": false
    },
    "release_date": {
      "displayed": true,
      "indexed": false,
      "ranked": false
    },
    "poster": {
      "displayed": true,
      "indexed": false,
      "ranked": false
    }
  }
}
