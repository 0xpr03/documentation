# Stats

# Get stat of an index

GET
/stats/:index

Get stats of an index.

# Path Variables

Variable Description
index The index UID

# Example

curl \
  --request GET 'http://localhost:8080/stats/12345678'

# Response: 200 Ok

{
  "numberOfDocuments": 19654,
  "isIndexing": false,
  "fieldsFrequency": {
    "poster": 19654,
    "release_date": 19654,
    "title": 19654,
    "id": 19654,
    "overview": 19654
  }
}
