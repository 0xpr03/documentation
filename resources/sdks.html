# SDKs and Integrations

You can use MeiliSearch API wrappers in your favorite language. These libraries support all API routes.

  • Golang (opens new window)
  • JavaScript (opens new window)
  • Laravel Scout (opens new window)
  • PHP (opens new window)
  • Python (opens new window)
  • Ruby (opens new window)
  • Rust (opens new window)
  • Swift (opens new window)

# Front-End Tools

  • VuePress plugin (opens new window)
  • docs-searchbar.js (opens new window): a search bar integration for all kinds of documentation.

# Other Tools

  • docs-scraper (opens new window): a scraper tool to automatically read the content of your documentation and store it into MeiliSearch.
  • meilisearch-digital-ocean (opens new window): how to deploy a MeiliSearch instance on DigitalOcean. See this dedicated page to use it.

# Contributing

If you want to build a new integration for MeiliSearch, you are more than welcome to and we would be happy to help you!

We are proud that some of our libraries were developed and are still maintained by external contributors! ♥️

We recommend to follow these guidelines (opens new window) so that it will be easier to integrate your work.

Feel free to contact us (opens new window)! 🙂
